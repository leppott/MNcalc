Input file requirements - Fish IBI

The comma-separated file (CSV) needs to have all the required fields listed in the table below. Column names must match the spelling, symbology and capitalization scheme exactly as shown.

IMPORTANT: Make sure you are using the proper Operational Taxonomic Unit (OTU) and taxa traits/attributes for your metric calculations. These can be found in the Taxa Translator and Attribute tables.

Column Name  Data Type Domain (allowable entries) Description
Index_Name text MN_IBI_Fish Tells R which model to apply (MN_IBI_Bugs, MN_IBI_Fish, MN_BCG)
Index_Class text 1, 2, 3, 4, 5, 6, 7, 10, 11 Fish IBI class. 1=Southern Rivers, 2 = Southern Streams, 3 = Southern Headwaters, 4 = Northern Rivers, 5 = Northern Streams, 6 = Northern Headwaters, 7 = Low Gradient Streams, 10 = Southern Coldwater, 11 = Northern Coldwater
SampleID text NA Unique sample identifier (typically comprised of the site name, sample date, and replicate number).
TaxaID text Scientific name Taxon name from the selected Operational Taxonomic Unit (OTU)
N_Taxa numeric non-negative real number Number of individuals.
N_Anomalies numeric NA Number of individuals with Deformities, Eroded fins, Lesions, Tumors (DELT) anomalies
Distance_m numeric NA Distance fished in meters. This is used for the catch-per-unit-effort (CPUE) metric in class 6 and 7
DRAINSQMI numeric NA Drainage area (total watershed area) in square miles
Gradient_m/km numeric NA Gradient calculation based on elevations (from a DEM) at the up- and downstream ends of a 1000m arc bracketing our sampling reach, representing slope in m/km
Type text Composition: DS,DSS,PERC, MIN-T. Life History: PI,SL. Schooling: SCH Combination of Composition, Life History and Schooling. Composition: DSS = DarterSculpinSucker, DS = DarterSculpin, EX = Exotic, MIN-T = Minnow - Tol, PERC = Perciformes. Life History: PI = Pioneer, SL = Short-lived. Schooling: SCH = schooling fish (Emerald Shiner and American Gizzard Shad)
Habitat text HW-T, WE-T, CW, CWN HW-T = Headwater Specialist, Exclude Tolerant, WE-T = Wetland, Exclude Tolerant, CW = Coldwater, CWN = Coldwater Native
Reproduction text MA>3-T,MA<2,NE,SILI,SER MA<2 = Mature Age < 2, MA>3-T = Mature Age > 3, Exclude Tolerant, NE = Non Lithophilic Nester, SER = Serial Spawner, SILI = Simple Lithophil
Toler text I,ICW,S,SCW,T,TCW,VT I = Intolerant, ICW = Intolerant Coldwater, S = Sensitive, SCW = Sensitive Coldwater, T = Tolerant, TCW = Tolerant Coldwater, VT = Very Tolerant
Trophic text BI-T,DE,DEM,GE,HE,IN-T,INCYP,OM,PI BI-T = Benthic Insectivore, Exclude Tolerant, DE = Detritivore, DEM = Detritivore Minor, GE = Generalist Feeder, HE = Herbivore, IN-T = Insectivore, Exclude Tolerant, INCYP = Insectivorous Cyprinid, OM = Omnivore, PI = Piscivore
