Input file requirements - Bug BCG

The comma-separated file (CSV) needs to have all the required fields listed in the table below. Column names must match the spelling, symbology and capitalization scheme exactly as shown.

IMPORTANT: Make sure you are using the proper Operational Taxonomic Unit (OTU) and taxa traits/attributes for your metric calculations. These can be found in the Taxa Translator and Attribute tables.

Column Name  Data Type Domain (allowable entries) Description
Index_Name text MN_BCG Tells R which model to apply (MN_IBI_Bugs, MN_IBI_Fish, MN_BCG)
Index_Class text bug1, bug2, bug3, bug4, bug5, bug6, bug7, bug8, bug9 Bug BCG class. bug1 = n forest rivers, bug2 = prairie rivers, bug3 = n riffle run (forest), bug4 = n forest glide-pool, bug5 = s riffle-run, bug6 = s forest glide-pool, bug7 = prairie glide-pool, bug8 = cool transitional, bug9 = s coldwater
GP.RR text GP,RR Riffle/Run (RR) if a stream reach includes riffles as representative habitat, is high gradient and has flow adequate to create an environment supportive of riffle dwelling organisms, then a stream would be considered. Glide/pool (GP) if these conditions are not met (for more details, see Table 6 in MPCA 2017 – macroinvertebrate protocols).
SampleID text NA Unique sample identifier (typically comprised of the site name, sample date, and replicate number)
TaxaID text NA Taxon name, using the BCG Operational Taxonomic Unit (OTU)
N_Taxa numeric non-negative real number Number of individuals
LargeRareCount numeric NA Count from the Large Rare pick. Taxa that only occur in Large Rare picks have blank entries in the N_Taxa column
BCG_Attr_GP text 1, 2, 3, 4, 5, 6 BCG taxa attribute assignment for Glide/Pool streams not in the bug 8 or bug 9 class. 1 = specialist or regionally endemic; 2 = highly sensitive; 3 = sensitive ubiquitous; 4 = moderately tolerant; 5 = tolerant; 6 = nonnative
BCG_Attr_RR text 1, 2, 3, 4, 5, 6 BCG taxa attribute assignment for Riffle/Run streams not in the bug 8 or bug 9 class. 1 = specialist or regionally endemic; 2 = highly sensitive; 3 = sensitive ubiquitous; 4 = moderately tolerant; 5 = tolerant; 6 = nonnative
BCG_Cool_8 text 1, 2, 3, 4, 5, 6 BCG taxa attribute assignment for bug8 (cool transitional) streams. 1 = specialist or regionally endemic; 2 = highly sensitive; 3 = sensitive ubiquitous; 4 = moderately tolerant; 5 = tolerant; 6 = nonnative
BCG_Cold_9 text 1, 2, 3, 4, 5, 6 BCG taxa attribute assignment for bug9 (s coldwater streams). 1 = specialist or regionally endemic; 2 = highly sensitive; 3 = sensitive ubiquitous; 4 = moderately tolerant; 5 = tolerant; 6 = nonnative
