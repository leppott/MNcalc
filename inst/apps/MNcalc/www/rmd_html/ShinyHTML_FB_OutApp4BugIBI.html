Input file requirements - Bug IBI

The comma-separated file (CSV) needs to have all the required fields listed in the table below. Column names must match the spelling, symbology and capitalization scheme exactly as shown.

IMPORTANT: Make sure you are using the proper Operational Taxonomic Unit (OTU) and taxa traits/attributes for your metric calculations. These can be found in the Taxa Translator and Attribute tables.

Column Name  Data Type Domain (allowable entries) Description
Index_Name text MN_IBI_Bugs Tells R which model to apply (MN_IBI_Bugs, MN_IBI_Fish, MN_BCG)
Index_Class text 1, 2, 3, 4, 5, 6, 7, 8, 9 Bug IBI class. 1=Northern Forest Rivers, 2 = Prairie Forest Rivers, 3 = Northern Forest Streams RR, 4 = Northern Forest Streams GP, 5 = Southern Stream RR, 6 = Southern Forest Streams GP, 7 = Prairie Streams GP, 8 = Northern Coldwater, 9 = Southern Coldwater
GP.RR text GP,RR Riffle/Run (RR) if a stream reach includes riffles as representative habitat, is high gradient and has flow adequate to create an environment supportive of riffle dwelling organisms, then a stream would be considered. Glide/pool (GP) if these conditions are not met (for more details, see Table 6 in MPCA 2017 – macroinvertebrate protocols).
SampleID text NA Unique sample identifier (typically comprised of the site name, sample date, and replicate number)
TaxaID text NA Taxon name, using the IBI Operational Taxonomic Unit (OTU)
N_Taxa numeric non-negative real number Number of individuals
LargeRareCount numeric NA Count from the Large Rare pick. Taxa that only occur in Large Rare picks have blank entries in the N_Taxa column
DRAINSQMI numeric NA Drainage area (total watershed area) in square miles. A drainage correction (continuous metric scoring adjustment) is applied to four of the metrics in the Southern Coldwater Stream MIBI, stream type 9.
TolVal numeric number ranging from 0-10, with decimals Tolerance values based on analyses of MPCA data. Used in several metric calculations (Hilsenhoff Biotic Index, intolerant taxa and tolerant taxa metrics). Very Intolerant taxa have values less than or equal to 2. Very Tolerant taxa have values equal to or greater than 8. Calculated using the weighted average of a general disturbance measure where taxa relative abundance was the weighting factor. The general disturbance measure was the first principal component of a principal components analysis of six disturbance variables including Minnesota’s Human Disturbance Score (HDS), the Minnesota Stream Habitat Assessment score, total phosphorus, total suspended solids, NH4, and nitrate/nitrite
TolVal2 numeric number ranging from 0-10, with decimals Coldwater sensitivity values based on analyses of MPCA data. Calculated using the weighted average of stream temperatures where taxa relative abundance was the weighting factor
LongLived logical TRUE, FALSE TRUE = macroinvertebrates with a life cycle of more than 1 year (i.e., semivoltine)
FFG text CG, CF, PR, SC, SH, MH, OM, PA, PH, XY Functional Feeding Group (method of food acquisition and functional role in aquatic food webs). Multiple entries are allowed and should be separated with a comma, either with or without a space (for example, either ‘CG,CF’ or ‘CG, CF’ are ok). Entries: collector-gatherer (CG), collector-filterer (CF), predator (PR), scraper (SC), shredder (SH), macrophyte herbivore (MH), omnivore (OM), parasite (PA), piercer-herbivore (PH), xylophage (XY)
Habit text BU, CB, CN, SP, SW Habit (how a macroinvertebrate moves in the aquatic environment and where they find food). Multiple entries are allowed and should be separated with a comma, either with or without a space (for example, either ‘BU,SP’ or ‘BU, SP’ are ok). Entries: Burrowers (BU), Climbers (CB), Clingers (CN), Sprawlers (SP), and Swimmers (SW)
Phylum text NA Taxonomic hierarchy
SubPhylum text NA Taxonomic hierarchy
Class text NA Taxonomic hierarchy
SubClass text NA Taxonomic hierarchy
Order text NA Taxonomic hierarchy
SubOrder text NA Taxonomic hierarchy
Family text NA Taxonomic hierarchy
SubFamily text NA Taxonomic hierarchy
Tribe text NA Taxonomic hierarchy
Genus text NA Taxonomic hierarchy
SubGenus text NA Taxonomic hierarchy
