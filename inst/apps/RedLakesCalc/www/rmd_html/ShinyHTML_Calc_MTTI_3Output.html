A single zip file will be available for download. It will include all result files along the the original data file. All result data files will be CSV format.

The zip file will included the base name of the original file along with the date and time.

The individual files will include the same basename along with a suffix to represent each step of the calculation process.

The entire criteria file is included to allow the user to check for non-matching Index Name and/or criteria columns.

As an example if the input file was “xxx.csv” the result files will look like those in the table below.

Calculation, MTTI, zip file contents.
Output file Description
xxx original input file
xxx_MTTI_flags_1_metrics.csv Metrics needed for flags. Including min and max MTTI values per sample taxa.
xxx_MTTI_flags_2_eval_long.csv results of flag calculations for all metrics and samples
xxx_MTT_flags_3_eval_summary.csv summary of how many samples had flagged metrics (and not flagged) based on the input file; count of # flagged (flag) and not flagged (NA)
xxx_MTTI_RESULTS.csv for each sample, MTTI model output plus flags (flag or NA)
xxx_MTTI_RESULTS.html boxplot showing distribution of samples across MTTI values.
