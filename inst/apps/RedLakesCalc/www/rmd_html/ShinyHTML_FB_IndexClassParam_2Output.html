A single zip file will be available for download. It will include all result files along the the original data file. All result data files will be CSV format.

The zip file will included the base name of the original file along with the date and time.

The individual files will include the same basename along with a suffix to represent each step of the calculation process.

The entire criteria file is included to allow the user to check for non-matching Index Name and/or criteria columns.

As an example if the input file was “xxx.csv” the result files will look like those in Table 1 below.

The results file will retain all columns from the input file. Additional fields will be added from the StreamCat and NHDplus datasets as described in the Table 2 below.

Table 1. File Builder, Generate Index_Class Parameters, zip file contents.
Output file Description
xxx Original input file
xxx_indexclassparam_RESULTS Original input file with additional fields appended. This file will be ready to run the Assign Index Class function in the File Builder section of the BCGcalc Shiny app.




Table 2. File Builder, Generate Index_Class Parameters, result file columns.
Column Description
COMID NHDplus COMID derived from StreamCat using latitude, longitude, and EPSG values provided by the user.
<existing columns> Columns from input data. Latitude and Longitude fields will be moved to after COMID.
EPSG_CALC EPSG value used to generate COMID
WSAREASQKM Total watershed area (km2) based on EPA StreamCat dataset (Hill et al. 2016) (not exact watershed delineation)
elev_m Mean elevation (meters) of local catchment area (km2) based on EPA StreamCat dataset (Hill et al. 2016) (not exact watershed delineation)
pslope_nhd % NHDPlusV2 flowline slope (slope*100)
slopelenkm length of NHDPlusV2 flowline (which the slope is based on)
gnis_name waterbody name based on NHDPlusV2
streamorde Strahler stream order. Source: NHD+ VAA file
ftype Flowline type based on NHDPlusV2 (e.g., stream/river, artificial path, connector/ditch)
fcode Flowline code based on NHDPlusV2
