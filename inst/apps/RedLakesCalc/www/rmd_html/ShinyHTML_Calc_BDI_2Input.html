Required Fields

The following fields are required in input files. However, the names can be different. The user will select the matching column names on screen before calculation.

Click here to download an Excel file with this table plus example data.

Table 1. Required fields.
Column Name Data Type Allowable entries Comment
Index_Name text BCG_PacNW_L1 OPTIONAL. (Will be added if not present). Directs the R code to the correct overall model to use (it references the INDEX_NAME column in the Rules table in the BCGcalc extdata folder)
Index_Class text ALL OPTIONAL. (Will be added if not present). Directs the R code to the correct submodel to use (it looks in the INDEX_CLASS column in the Rules table in the BCGcalc extdata folder)
SAMPLEID text NA Unique sample identifier. User can select the proper column if spelling doesn’t match.
TAXAID text NA Taxonomic identification of organisms in each sample. User can select the proper column if spelling doesn’t match.
N_TAXA numeric non-negative real number Number of individuals. User can select the proper column if spelling doesn’t match.
EXCLUDE text TRUE, FALSE Redundant (non-distinct) taxa should marked “TRUE”. Redundant taxa are excluded from richness metric calculations but are included in the composition (percent individuals) metrics. User can select the proper column if spelling doesn’t match.
NONTARGET logical TRUE, FALSE Non-target taxa should be marked “TRUE”. The R code excludes non-target taxa from all metric calculations.
PHYLUM text NA taxonomic hierarchy
CLASS text NA taxonomic hierarchy
SUBCLASS text NA taxonomic hierarchy
ORDER text NA taxonomic hierarchy
FAMILY text NA taxonomic hierarchy
SUBFAMILY text NA taxonomic hierarchy
TRIBE text NA taxonomic hierarchy
GENUS text NA taxonomic hierarchy
BCG_ATTR text 1i, 1m, 2, 3, 4, 5, 6 Biological Condition Gradient (BCG) attribute assignments
THERMAL_INDICATOR text stenoc, cold, cool, cowa, warm, stenow, eurythermal OPTIONAL. Thermal preference assignment based on Stamp et al. (in progress): stenoc=cold stenotherm; cold, cool, cowa = cool/warm, warm, stenow = warm stenotherm, eurythermal.
FFG text CG, CF, PR, SC, SH, MH, OM, PA, PH, XY Primary Functional Feeding Group. Multiple entries are allowed and should be separated with a comma (and no space; e.g., CG,CF). Entries: collector-gatherer (CG), collector-filterer (CF), predator (PR), scraper (SC), shredder (SH), macrophyte herbivore (MH), omnivore (OM), parasite (PA), piercer-herbivore (PH), xylophage (XY).
LONG_LIVED logical TRUE, FALSE Is the taxon long-lived? TRUE or FALSE
NOTEWORTHY logical TRUE, FALSE Is the taxon “noteworthy”? TRUE or FALSE
